# Home Cleaning Website

## Deployment Instructions for GitHub Pages
1. In your repository on GitHub, navigate to Settings.
2. Under the "Pages" section on the left sidebar, select the source of your GitHub Pages site. You can choose to serve from the `main` branch, root folder or `/docs` folder.
3. After selecting the source, click "Save". Your site will now be published.
4. Once published, the URL to your GitHub Pages site will be displayed in the "GitHub Pages" section.

## Formspree Integration
To handle form submissions via Formspree:
1. Sign up at [Formspree.io](https://formspree.io/).
2. Create a new form in your dashboard and receive the endpoint URL for form submission.
3. In your website's contact form, set the `action` attribute to the Formspree URL received earlier and ensure the method is set to POST:
   ```html
   <form action="YOUR_FORMSPREE_URL" method="POST">
   ```
4. Add any relevant fields to your form; Formspree will handle the submissions.

## SEO Checklist for Home Cleaning Business Website
- [ ] **Transparent contact information:** Ensure your contact details are easy to find.
- [ ] **Relevant Keywords:** Use keywords related to home cleaning (e.g., "home cleaning services", "eco-friendly cleaning").
- [ ] **Content Quality:** Create content that answers common questions or concerns of your clients.
- [ ] **Meta Tags:** Use descriptive and keyword-rich meta titles and descriptions for each page.
- [ ] **Mobile Optimization:** Ensure that your site is responsive on mobile devices.
- [ ] **Social Media Links:** Add links to your social media channels.
- [ ] **Analytics Setup:** Use Google Analytics to track your website's performance.
- [ ] **Local SEO:** If applicable, include your location in your site and service descriptions.
- [ ] **Quality Backlinks:** Aim to have backlinks from reputable websites to enhance domain authority.
- [ ] **Page Speed Optimization:** Ensure your website loads quickly to reduce bounce rates.
